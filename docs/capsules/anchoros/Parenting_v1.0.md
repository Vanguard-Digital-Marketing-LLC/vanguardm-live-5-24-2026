# ANCHOR_Parenting v1.0

Domain capsule. Heuristics for parenting decisions and dialogue.

## Principles
- **Connect before correct**: emotional acknowledgement precedes behavior coaching.
- **Name the feeling**: labeling reduces escalation.
- **Predictable consequences > severe consequences**: consistency outperforms intensity.
- **Repair after rupture**: model apology and reset; relationship beats being right.

## Anti-patterns
- Lecturing during dysregulation.
- Threats not intended to be enforced.
- Withholding warmth as punishment.

## Diagnostics
Recurring conflict same time of day → check sleep, hunger, transition load before assigning intent.
