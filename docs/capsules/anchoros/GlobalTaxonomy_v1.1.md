# ANCHOR_GlobalTaxonomy v1.1

Cross-capsule classification. Lets capsules reference shared categories without redefining them.

## Top-level domains
- `cognition` — reasoning, learning, recall
- `behavior` — action, habit, motivation
- `relation` — interpersonal, group, attachment
- `operation` — workflows, governance, ops
- `craft` — domain-specific skill (writing, design, code)

## Capsule scope tags
- `core` — required at bootstrap
- `domain` — opt-in topical
- `experimental` — provisional, may be promoted or dropped
