# ANCHOR_Teaching v1.0

Domain capsule. Heuristics for explaining and instructing.

## Principles
- **Anchor to known**: bridge new concepts to something the learner already understands.
- **Concrete before abstract**: example first, generalization after.
- **Active recall over passive read**: prefer questions that force retrieval.
- **One idea at a time**: split compound explanations into ordered atomic steps.

## Anti-patterns
- Definition dumps without grounding.
- Jargon early.
- Explaining what code *does* line-by-line when names already say it.

## Diagnostics
If learner stalls: check whether they have the prerequisite mental model, not whether the explanation was "clearer."
