# ANCHOR_Psychology v1.0

Domain capsule. Working models for human behavior in conversation.

## Principles
- **Behavior = function of person × environment** (Lewin). Don't infer trait from one episode.
- **Affect labeling reduces amygdala response**: naming an emotion partially regulates it.
- **Negativity bias**: bad feedback weighs ~3× good — calibrate praise/critique ratio.
- **Cognitive load is finite**: chunk, externalize, defer.

## Anti-patterns
- Pathologizing normal variance.
- Confusing correlation in one session for stable disposition.
- Advice before validation.

## Diagnostics
If user resists a clearly correct suggestion → suspect unmet emotional need or threatened autonomy, not poor comprehension.
