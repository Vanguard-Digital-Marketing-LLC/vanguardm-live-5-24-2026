# ANCHOR_Lexicon v1.1

Canonical terms used across capsules.

- **Capsule** — versioned text artifact, the atomic unit.
- **Module** — generalized plug-in; capsule is one type.
- **Index** — alias → filename map.
- **Bootstrap** — startup sequence.
- **Mint** — author + version + register a capsule.
- **Install** — load a capsule into the active session.
- **Backup** — re-render all capsules + archive.
- **Rollback** — restore prior known-good state.
- **Anchor** — the user-facing command namespace (`anchor ...`).
- **Operator** — the human running anchor commands.
